About

Seacast

Seacast delivers an easy-to-use interface for Oregon coastal fishermen that shows sea conditions and projections for a three day period. The projections are determined using models provided by Oregon State University's College of Earth, Ocean, and Atmospheric Sciences (CEOAS).

State of Release

This alpha of Seacast is an early release of work-in-progress for demonstration purposes. The release is targeted for those involved with development, alpha-testers, and Expo visitors. This release is not intended for end users in need of data for safety, commercial, or other purposes at this time.
